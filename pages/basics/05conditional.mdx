But what if the condition inside the "if" statement is false? That's where the "else" statement comes to the rescue! It allows us to specify a block of code to be executed when the condition is false. ğŸ˜

Here's an example:

```java

    System.out.println("Good day.");
} else {
    System.out.println("Good evening.");
}
```

Since the value of `time` is 20, which is greater than 18, it will print "Good evening." ğŸŒ™

